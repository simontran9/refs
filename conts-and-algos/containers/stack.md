# Stack

## Operations

| **Operation**     | **Description**                                       |
| ----------------- | ----------------------------------------------------- |
| `func top() -> E` | Returns the topmost element in the stack.             |
| `func push(e: E)` | Adds an element to the top of the stack.              |
| `func pop() -> E` | Removes and returns the topmost element in the stack. |
